## 1: Clone the project
```
git clone git@github.com:mchaaar/cyna-web.git
```

```
cd cyna-web
```

## 2: Add these local lines to the .env

```.env
NEXT_PUBLIC_API_URL=http://localhost:8000
NEXT_PUBLIC_APP_NAME=Cyna
NEXT_PUBLIC_APP_ENV=development
```

## 3: Install dependencies
```
npm install
```

## 4: Start the project
```
npm run dev
```
